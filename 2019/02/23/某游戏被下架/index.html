已下架的某游戏

发表于 2019-02-23 | 更新于 2019-02-24 | 分类于 随想

这件事出来后很多人感到被冒犯，而我更多是觉得惋惜和无奈。

有自己的政治立场很正常，甚至说，我觉得这才正常。但不分场合、不分途径、不分方式地表达，并非自由，是野蛮。夹带私货的做法无疑践踏了游戏背后所有人的心血，也让众多喜欢这款游戏的玩家失落，甚至会给业界带来不得而知的影响。而付出如此代价换来的诉求，却注定得不到正视和尊重。

所以，我憎恨做出此举动的人（或团体）。但我不为游戏下架叫好，更不想看两边网民的争吵。我只感到惋惜和无奈。

下架力度很大，比我预想中的，或者说比愿望中的更大。下午不时在想这样的做法是否妥当，会不会有更好的处理方法，但也得不出什么头绪。脑里更多的，还是对优质游戏落幕的遗憾吧。
